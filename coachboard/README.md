# Football Tactic Board

A deployable static HTML, CSS, and JavaScript football tactics board.

## Files

- `index.html`
- `styles.css`
- `script.js`

## Features

- Responsive layout for desktop, iPad, and iPhone
- Full field, attacking half, left corner, right corner, and free kick zone views
- Drag home and away players
- Formation presets: 4-3-3, 4-4-2, 4-2-3-1, 3-5-2, and set piece shape
- Draw arrows and lines
- Change home team, away team, drawing, and ball colors
- Place, drag, and hide a ball
- Toggle home and away teams
- Add a coaching note
- Flip the board
- Clear drawings or reset the board
- Expand the current field view into fullscreen display
- Export board as a PNG

## Deployment

Upload `index.html`, `styles.css`, and `script.js` to the same folder on your website.

For GitHub Pages, place them in your repository root or `/docs` folder, then enable Pages in repository settings.
